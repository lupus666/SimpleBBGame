- [x] 支持动画

## 游戏启动
  
  由于目前仅是单机版，直接运行`BBG.exe`文件便可运行

## 游戏玩法

 - 与球球大作战玩法相同
    
### 游戏操作
 - 点击鼠标左键进行移动操作（存在BUG）
 - 点击键盘D键进行分裂操作
 - 点击键盘S键进行吐孢子操作
